Research Topic

Formal methods for Cyber Security

Cloud computing plays an important role in revolutionizing the computational infrastructure used by modern business models, as it provides an on-demand resources on a cost effective way. To fulfill these demands, there are significant number of companies competing to provide the best cloud services. However, regardless of its cost benefits, customers still lack confidence in cloud services due to security issues it poses.The deficiencies of existing approaches and the proliferation of security bugs has grown the interest in the use of formal methods to validate the security of vulnerable systems. Consequently, the literature on the use of formal methods for cyber security has grown in the past decade. Formal method approaches offers several desirable properties. Unlike traditional defenses, it provides a guarantee on certain aspects of the system which are properly validated. However, prior proposed techniques either require manual analysis or do not consider cloud realistic attack scenarios. In this proposal we consider realistic attack scenarios from CAPEC database and generate abuse cases to derive formal security specification for cloud computing services.
